# BookStore-API

.NET Core 3.1 Book Store Web API for educational purposes. This project implements:<br />
*.NET Core 3.1<br />
*AutoMapper<br />
*NLog<br />
*Swagger UI<br />
*Entity Framework Core<br />
*Repository Pattern<br />
*Identity and Authentication<br />
*Security with JSON Web Token

<br />
